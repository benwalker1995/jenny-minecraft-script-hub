## What this mod does

Jenny Companion Mod is a Minecraft Forge 1.20 addon built around an NPC companion that can remain available across multiple play sessions and react to the player in a more lifelike way. The core design combines dialogue memory, emotion-based responses, and multilingual conversation handling so the companion can better fit different users and language preferences.

Beyond chat, the mod also offers a video diary system, behavior that changes at night, and modular switches that let you choose which pieces stay active.
